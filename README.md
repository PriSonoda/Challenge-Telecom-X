# Challenge-Telecom-X
Análise de Evasão de Clientes (Churn)

📌 Visão Geral
Este projeto tem como objetivo analisar os fatores que influenciam a evasão de clientes (churn) em uma empresa de telecomunicações. A análise inclui limpeza de dados, exploração visual, identificação de padrões e recomendações baseadas nos resultados.

🗂️ Estrutura do Projeto
- Notebook de Análise (Google Colab): contém todo o processo de limpeza, exploração e visualização dos dados.
- Dataset: base de clientes com informações contratuais, demográficas e financeiras.
- Relatório Final: resumo dos principais achados e recomendações.

🛠️ Instalação e Dependências
Para executar o notebook, é necessário ter instalado:
- Python 3.x
- Pandas
- NumPy
- Matplotlib
- Seaborn
No Google Colab, todas essas dependências já vêm instaladas.

▶️ Como Executar o Projeto
- Abra o notebook no Google Colab.
- Execute as células na ordem apresentada.
- As visualizações serão geradas automaticamente.
- O relatório final está ao final do notebook.

🔍 O que foi feito na análise
- Limpeza e padronização dos dados
- Criação de novas variáveis
- Visualizações exploratórias
- Comparação entre clientes que cancelaram e não cancelaram
- Identificação de padrões de churn
- Recomendações práticas para reduzir evasão

📊 Principais Resultados
- A taxa de churn é de aproximadamente 26,5%.
- Clientes com contrato mensal têm maior probabilidade de cancelar.
- Clientes com pouco tempo de contrato cancelam mais.
- Cobranças mensais mais altas estão associadas ao churn.
- O gênero não influencia a evasão.
- Clientes com internet fibra apresentam maior churn.

💡 Conclusões
A evasão está relacionada principalmente a fatores contratuais e financeiros, não demográficos.
As ações mais eficazes envolvem:
- melhorar o onboarding nos primeiros meses;
- incentivar contratos mais longos;
- revisar preços de planos mais caros;
- investigar problemas no serviço de fibra;
- criar modelos de previsão de churn.
